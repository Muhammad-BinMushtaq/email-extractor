## üéì Key Concepts

### Authentication
- **Google Sign-In**: Email + name signup
- **Admin Access**: Password-protected (default: 123456)
- **Session Storage**: localStorage persists user data

### Features
- **Scraper**: Extract emails from websites (3/day free, unlimited premium)
- **Composer**: Generate professional emails (9 templates)
- **Campaign**: Send bulk emails via Gmail SMTP
- **Saved Drafts**: Store and reuse email templates
- **Admin Dashboard**: Manage payments and users

### Monetization
- **Free Tier**: 3 searches/day
- **Premium**: Unlimited searches + perks (Rs. 499/month)
- **Payment**: EasyPaisa with admin approval
- **Revenue**: Tracked in admin dashboard

## ‚ùì Frequently Asked Questions

### General
**Q: How much does EmailPro cost?**  
A: Free (3 searches/day), Premium (Rs. 499/month for unlimited)

**Q: Can I use my regular Gmail password?**  
A: No, you must use an App-Specific Password. See USER_GUIDE.md

**Q: How is my data stored?**  
A: Currently in browser localStorage. Future: PostgreSQL database

**Q: Can I cancel my premium subscription?**  
A: Contact admin via GitHub issues (feature coming soon)

### Technical
**Q: What if emails don't scrape from a website?**  
A: Some sites block scrapers. Check the website is accessible. See USER_GUIDE.md Troubleshooting

**Q: Why can't I send emails?**  
A: Most likely Gmail authentication issue. See USER_GUIDE.md Email Sending Tips

**Q: How do I report a bug?**  
A: Open an issue on GitHub: https://github.com/Muhammad-BinMushtaq/email-extractor/issues

**Q: Can I contribute to the project?**  
A: Yes! Fork the repo and submit a pull request.
